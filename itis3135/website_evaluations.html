Web Evaluations

Listed below are two different websites. One has been deemed a good website, and one has been deemed a bad website. How the websites have been judged is based on the CRAP principle. CRAP is an acronym and it stands for:

  • Contrast
  • Repetition
  • Alignment
  • Proximity

Each website evaluation breaks down the website by the CRAP principle, so you can see how we judged the website.


Good Website - MTGGoldfish

MTGGoldfish is a website aimed at providing updates on the economy and strategy of the card game "Magic: The Gathering". Here's why we think the website is good:

  • Contrast: While the website does use solid white and black for some elements, it uses orange for it's menu items. This blends well with the contrasting white and black, as well as the blue link colors.
  • Repetition: The website utilizes a consistent color scheme, and most if not all elements on any given page are sized based on their category (articles, products, tools, etc.)
  • Alignment: The website does very well at maintining a consistent "centered" alignment. What this means is that the elements are symmetrical, but the text isn't unneccessarily centered. On pages with side elements, like the "decks" page, it maintains a similar array style while giving just the right amount of room to the other elements.
  • Proximity: The website shows good attention to proximity, where similar items are grouped very close together, while maintining consinstent spacing between the different groups of items so as to not confuse the reader. This is good categorization by proximity.
